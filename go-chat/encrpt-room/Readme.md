# 암호화 채팅 방법 1. 방전용 대칭키 
✅ 구조 요약
✅ 서버	방에 처음 참가자가 들어오면 AES 키 생성 → 저장
✅ 서버	이후 입장하는 사람에게는 공개키 기반으로 AES 키를 암호화해서 전송
✅ 클라이언트	개인 키 생성 후 서버에 공개키 전송
✅ 클라이언트	서버로부터 암호화된 AES 키를 수신하고 복호화
✅ 클라이언트	이후 메시지를 roomAESKey로 암호화하여 전송
